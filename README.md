# ExcuseMe Social Planner App

## Pages
1. / (public) - homepage explaining app
2. / (signed in) - homepage showing upcoming events
3. / (profile) - shows profile, sign out option n points
4. /groups - create new groups and add people
5. /event/[id] - view event and give excuses
